# 🎓 Internship Feedback Management System

A comprehensive portal to manage, evaluate, and track student internships across institutions. This platform provides responsive, dynamic, and role-based access for Students, Faculty, Companies, and Administrators.

## ✨ Features by Role

### 👨‍🎓 Student Portal
- **Dashboard:** Visually track internship milestones, pending tasks, and recent notifications.
- **Approval Tracking:** Upload documents (Offer Letters, NOCs) and monitor their real-time approval status.
- **Feedback Logs:** Easily access, draft, and submit periodic reflection logbooks.

### 👨‍🏫 Faculty Portal
- **Dashboard Tracking:** High-level overview of active assigned students and pending evaluations.
- **Evaluation Rubrics:** Interactive split-view grading tables integrated seamlessly with company mentor feedback context.
- **Document Verification:** Immediate inline actions (Approve/Reject) for incoming student uploads.

### 🏢 Company Portal
- **Feedback Forms:** Clean, highly interactive slider-based forms for accurately assessing student performance.
- **Automated Summaries:** Streamlined submission workflows targeting Technical Ability, Collaboration, and Reliability.

### 👑 Admin Portal
- **Global Reports:** Comprehensive timeline charts, bar charts, and program-wide performance statistics.
- **User Management:** Instant provisioning, editing, and disabling of roles directly via an intuitive datatable.
- **Template Gallery:** Distribute and manage dynamic assessment frameworks distributed to all reporting supervisors.

2. **Start the local development server:**
   ```bash
   npm run dev
   ```
